Coffee Machine Program

This is a console-based coffee machine program written in Python. Users can select from espresso, latte, or cappuccino, and the program handles resource management, coin transactions, and drink preparation.

Features:
Prompt the user for a drink choice: espresso, latte, or cappuccino. 
Secret command off to turn off the machine. 
report command to display current resources (water, milk, coffee, money). 
Checks if there are enough ingredients to make the selected drink. 
Processes coins: quarters, dimes, nickels, and pennies. 
Checks if the user has inserted enough money and gives change if necessary. 
Deducts resources when a drink is made and updates profit. 
Displays messages when drinks are served or if resources are insufficient.
