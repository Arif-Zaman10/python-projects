Turtle Sketch Controller
A simple, interactive Python program that allows you to control a turtle on the screen using your keyboard. This project uses the built-in turtle module to create a basic "Etch-A-Sketch" style drawing application.

🎮 Controls
Use the following keys on your keyboard to interact with the turtle:

w: Move the turtle Forward by 50 units.

s: Move the turtle Backward by 50 units.

a: Turn the turtle Counter-Clockwise (Left) by 50 degrees.

d: Turn the turtle Clockwise (Right) by 50 degrees.

c: Clear the screen and return the turtle to the home position.
