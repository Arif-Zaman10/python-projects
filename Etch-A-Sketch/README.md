Turtle Keyboard Controller
A simple interactive drawing program built with Python's turtle module. Control a turtle on screen using your keyboard – move, turn, and clear the canvas with ease.

Controls
Key	Action
W	Move forward 50 pixels
S	Move backward 50 pixels
A	Turn left 50 degrees
D	Turn right 50 degrees
C	Clear screen and return to home
How to Run
Make sure you have Python installed (version 3.x).

Save the provided script (from the source) in a file, e.g., turtle_controller.py.

Open a terminal or command prompt in that folder.

Run the command:

bash
python turtle_controller.py
Click on the turtle graphics window and press the keys above to control the turtle.

Requirements
Python 3 (the turtle module is included in the standard library – no extra installation needed)

Example
After launching, press W a few times to draw a line, then D to turn, and W again to create a simple shape. Press C to erase everything and start fresh.

Notes
The program listens for key presses only while the graphics window is active.

Close the window by clicking on it or pressing the close button.
